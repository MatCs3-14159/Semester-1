NetPlix
(API - OMDb API - The Open Movie Database)

NetPlix is a modern, visually rich movie discovery web application designed to provide a smooth and engaging user experience. The app allows users to search for movies in real time using the OMDb open API, displaying essential details such as movie posters, titles, and release years. It features predefined movie categories including Popular, Action, Fantasy, and Sci-Fi movies, which are automatically loaded when the page opens. Built using asynchronous JavaScript and the Fetch API, the application retrieves data efficiently without page reloads. The responsive layout, animated movie cards, stylish navigation bar, and intuitive search interface enhance usability across both desktop and mobile devices, making movie exploration fast, interactive, and enjoyable.
